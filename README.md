# מטלה מתוקשבת — שם המספר

האתר נבנה לפי קובץ ה־Word שהועלה בנושא שם המספר.

כולל:
- דף שער באותו סגנון של האתר הקודם.
- נושא חדש: שם המספר.
- בחירת כיתה ושם מורה מתוך מאגר.
- שאלה 1 בנויה כתשובות מרובות / בחירה מרובה.
- כל שאלה בדף נפרד.
- ציון סופי מתוך 30 נקודות.
- שמירת תשובות וציונים ל־Google Sheets דרך Apps Script.
- בקרת חריגות: יציאה מהמסך, יציאה ממסך מלא, העתקה/הדבקה.
